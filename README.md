# Distillation-Column-Course-Design
适用于化工原理课程设计的精馏塔设计

按顺序执行中文命名程序即可

前两个英文程序必须与剩下的中文程序在同一文件夹中，否则2.1无法运行

本程序体系为苯-甲苯，其他体系需更改程序中的物性数据字典

塔板数计算依靠绘图实现，若手工绘图与本程序绘图所得塔板数不一致，以手工绘图为准，并更改相应.json的塔板数数据，以便数据正确传递给下一个程序
